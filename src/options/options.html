Security Notice

Your API key is stored securely on your device only (using Chrome's encrypted storage). It will never be synced or sent to any server except Google's Gemini API.

Important: Never share your API key. If you suspect it has been compromised, revoke it immediately in Google AI Studio.

Get your API key from Google AI Studio

How it works

  • Get your API key: Visit Google AI Studio to create a free API key
  • Secure storage: Your key is encrypted and stored locally on your device
  • Direct connection: The extension connects directly to Google's Gemini API
  • Key restrictions: You can restrict your API key to specific domains in Google Cloud Console

Key Rotation

If you need to rotate or revoke your API key:

  1. Go to Google AI Studio
  2. Revoke the old key if compromised
  3. Create a new key
  4. Update it here in the extension settings
